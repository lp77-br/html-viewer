Este repositório concentra três edições distintas de editores HTML para Android, cada uma focada em uma necessidade específica do desenvolvedor, desde a leveza extrema até a extração avançada de sites.

---

## 📂 Escolha sua Versão

### ⚡ [Android LITE](android-lite/)
Focada em minimalismo extremo e zero desperdício.
- **Diferencial:** Interface 100% via código Java (sem XML).
- **Ideal para:** Dispositivos antigos ou quem busca o APK mais leve possível.

### 🏠 [Android OFFLINE](android-offline/)
Uma IDE completa que funciona sem qualquer acesso à rede.
- **Diferencial:** Editor CodeMirror com suporte a abas HTML/CSS/JS.
- **Ideal para:** Codar com privacidade total e sem distrações.

### 🌐 [Android ONLINE](android-online/)
A ferramenta definitiva para Web Extraction.
- **Diferencial:** Motor de importação de URL que extrai o código de sites reais.
- **Ideal para:** Estudar estruturas de sites e sincronizar códigos via CDN.
