A versão **LITE** do nosso editor de HTML é focada em minimalismo extremo e "zero desperdício". Desenvolvida para ser rápida, simples e funcional.

## ✨ Diferenciais desta Versão
* **Interface via Código:** UI 100% construída em Java, sem arquivos XML de layout, garantindo carregamento instantâneo.
* **Auto-Save:** Seu código é salvo automaticamente no `SharedPreferences` ao rodar o preview.
* **Modo Fullscreen:** Aproveitamento total da tela para digitar e para ver o resultado.

| **Bibliotecas** | AppCompat & Design |
| **Assinatura APK** | Testkey (Debug) |

## 🚀 Como Compilar
Se você deseja modificar este código via **Sketchware Pro** ou **Android Studio**:
1. **Dependências:** Certifique-se de que as bibliotecas `AppCompat` e `Design` estão ativadas no seu ambiente.
2. **Java 8:** Configure o compilador para Java 1.8 para evitar erros de sintaxe e garantir compatibilidade com o motor D8.
